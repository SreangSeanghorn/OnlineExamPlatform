About TnakRan

TnakRan is a modern online examination and classroom platform designed to make learning more accessible and organized. We aim to bring a user-friendly, responsive, and reliable experience to both students and educators.

Core Features

  • Seamless class enrollment and scheduling
  • Interactive exams and real-time results
  • Notification system for events and deadlines
  • Secure authentication and user roles
